# makar_1alpha

Лендинг на `React + Vite`, подготовленный для публикации на GitHub Pages.

## Локальный запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```

## Публикация на GitHub Pages

Репозиторий: `https://github.com/Nikzoom/project1.0`

После пуша в ветку `main` сайт автоматически соберется и опубликуется через GitHub Actions.

Ожидаемый адрес сайта:

```text
https://nikzoom.github.io/project1.0/
```

Если нужен ручной деплой в ветку `gh-pages`, есть команда:

```bash
npm run deploy
```
